# 📌 Projeto 3: Aplicativo de Saúde Mental para Jovens

## 🚀 Sobre o Projeto
Este projeto é um aplicativo desenvolvido com Ionic + Angular para gerenciar um diário emocional, permitindo que os usuários registrem suas emoções e acompanhem sua evolução ao longo do tempo.

https://www.figma.com/community/file/1487607625714360477  


---

## 🛠️ Tecnologias Utilizadas
- **Ionic Framework**
- **Angular**
- **TypeScript**

---

## 📂 Estrutura do Projeto
```
/src
│── app/               # Código principal do aplicativo
│── environments/      # Configuração de ambiente
│── assets/            # Arquivos estáticos (ícones, imagens)
│── theme/             # Estilização do aplicativo
│── index.html         # Arquivo raiz do projeto
│── main.ts            # Ponto de entrada do aplicativo
│── polyfills.ts       # Configuração de compatibilidade
```

---

## ▶️ Como Rodar o Projeto

### **  Pré-requisitos**
Antes de iniciar, certifique-se de ter instalado:
- **Node.js** (versão 16 ou superior)
- **Ionic CLI** (caso não tenha, instale com `npm install -g @ionic/cli`) 

### **2  Instalar Dependências**
No diretório raiz do projeto, execute:

npm install


### **3  Rodar o Servidor de Desenvolvimento**
Para iniciar o aplicativo em ambiente local:

ionic serve

O comando abrirá o projeto no navegador em `http://localhost:8100/` 🚀
